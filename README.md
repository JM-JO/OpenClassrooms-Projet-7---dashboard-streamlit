# Projet-7---dashboard-streamlit

This repository contains the code for the dashboard.
  
It is deployment-ready for Heroku.

Deployed on Heroku: https://project7-dashboard-streamlit.herokuapp.com/

Please be patient when you try to reach the website, the server wake-up takes about 20 seconds (in the meantime, you may want to wake up the API server at https://project7-api-ml.herokuapp.com/ )
